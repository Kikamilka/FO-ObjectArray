1. Прочитайте подробно про Object.create. Создайте функцию "частичного" клонирования. На вход функция получает объект и список пропертей, которые должны быть сколнированы. Например, foo(obj, ["prop1", "prop2"]).

Остальные свойства, должны принадлежать клонируемому объекту.

Например,

let obj = {prop1: 1, prop2: { subprop: 1 } , prop3: { subprop: 1}}; let clone = foo(obj, ["prop2"]); clone.prop2.subprop = 3; // изменился только clone clone.prop3.subprop = 3; // изменились clone и obj

* Доработать функцию так, чтобы скалярные проперти, не указанные в списке, тоже оставались "связанными".

clone.prop1 = 3; // изменились clone и obj

* Сделать поддержку составных путей в именах проперей. Например, foo(obj, ["prop1.subprop[2].prop"]). Могут сильно помочь функции для работы с путями из lodash. (https://lodash.com/docs)
